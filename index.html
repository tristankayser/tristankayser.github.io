Local LLM · Agent Framework · Self-Hosted
Running a private AI assistant stack with Ollama serving large language models entirely on-device — no cloud, no data leaving the network. Hermes Agent sits on top as the persistent AI framework, enabling long-term memory, skill-building, and multi-platform messaging.

DNS sinkhole for the entire network. Blocks ads, trackers, and malicious domains before they reach any device.
